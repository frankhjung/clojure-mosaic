# Technology Stack

## Core Language & Runtime

- **Clojure (v1.12.4):** A dynamic, functional dialect of Lisp hosted on the JVM.
- **Java SE (v11+):** The underlying platform providing high-performance graphics and I/O.

## Libraries & Frameworks

- **Java AWT:** Used for low-level pixel manipulation, image resizing, and composition.
- **ImageIO:** Handles reading and writing of standard image formats (JPEG, PNG).
- **clojure.tools.cli:** Orchestrates the command-line interface and argument parsing.

## Build & Development Tools

- **Leiningen:** The project automation and dependency management tool.
- **cljfmt:** Ensures consistent code formatting across the project.
- **clj-kondo:** Performs static analysis and linting to catch errors early.
- **eftest:** A fast, parallel test runner for Clojure.

## Deployment & Execution

- **Uberjar:** Standard packaging format for standalone JVM applications.
